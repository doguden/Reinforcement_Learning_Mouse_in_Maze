
A reinforcement learnign example, perfect for teaching the concept. 
A mouse in a maze with asjustable difficulties.
The mouse can see his surroundings and have x and Y coordinates.
Periodic boundary on x and continuous on y, 30x120 squares.
The models starts with a basic NN but can be converted to more complex models.
<img width="1849" height="925" alt="image" src="https://github.com/user-attachments/assets/54cba1d2-676c-401f-aa8d-24a5b4b4c27f" />

Designed to study model convergence and hyperparameters.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
